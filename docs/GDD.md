# 🍞 My Little Teddy Bakery
귀엽고 힐링되는 동물 베이커리 리듬 타이쿤 게임

---

# 1. Game Overview

## 장르
Rhythm + Tycoon (완전 액티브형)

## 플레이 타겟
짧은 시간에 편하게 즐기는 캐주얼 모바일 유저

## 핵심 경험
리듬 게임을 통해 빵을 만들고 돈을 벌어 베이커리를 성장시키는 게임

---

# 2. Game Pillars (핵심 가치)

1. 힐링 감성 (스트레스 없음)
2. 짧은 플레이 세션 (20~30초)
3. 반복 플레이 동기
4. 단순하지만 성장하는 재미

---

# 3. Theme & World

## 세계관
귀여운 동물들이 운영하는 작은 베이커리

## 플레이어 캐릭터
🐻 곰 베이커리 사장

---

# 4. Core Game Loop

1. 리듬 미니게임 플레이
2. 빵 완성
3. 자동 판매
4. 돈 획득
5. 업그레이드 구매
6. 반복

플레이하지 않으면 돈을 벌 수 없음 (완전 액티브형)

---

# 5. MVP Scope (출시 버전 범위)

## 포함 기능
- 메인 화면 1개
- 리듬 미니게임 1종
- 빵 1종 (식빵)
- 자동 판매 시스템
- 업그레이드 3종

## 제외 기능 (업데이트 예정)
- 손님 시스템
- 여러 빵 종류
- 매장 확장
- 직원 시스템
- 스토리

---

# 6. Rhythm Mini Game Design

## 노트 종류
- Tap
- Hold
- Swipe

## 빵 제작 단계 매핑
| 단계 | 액션 |
|---|---|
| 반죽 섞기 | 빠른 탭 |
| 반죽 치대기 | Hold |
| 오븐 넣기 | 타이밍 탭 |
| 토핑 추가 | Swipe |

## 플레이 시간
20~30초

## 실패 패널티
게임 오버 없음  
품질만 감소

---

# 7. Scoring System

## 판정
| 판정 | 점수 |
|---|---|
| Perfect | 100 |
| Good | 50 |
| Miss | 0 |

## 품질 계산
품질 = 획득 점수 / 최대 점수

## 품질 → 가격 배율
| 품질 | 배율 |
|---|---|
| 95~100% | x1.5 |
| 80~94% | x1.2 |
| 60~79% | x1.0 |
| 0~59% | x0.5 |

---

# 8. Economy Design

## 기본 값
- 기본 빵 가격: 100
- 플레이 시간: 25초

---

# 9. Upgrade System

## 1️⃣ 레시피 개선
빵 기본 가격 +10%

## 2️⃣ 좋은 오븐
Perfect 판정 범위 증가

## 3️⃣ 대형 오븐
1판 생산량 증가 (1 → 5)

---

# 10. Screen Structure

단일 화면 구조

- 상단: 돈 / 업그레이드 버튼
- 중앙: 주방 + 캐릭터
- 중앙 버튼: 리듬 게임 시작
- 하단: 업그레이드 메뉴
